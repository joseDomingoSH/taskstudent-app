Aplicación para Estudiantes

Está aplicación es para que los estudiantes
tengan organizadas sus tareas semanales las
cuales tienen su siguiente estructura:

├── bd
│   └── tareas.txt
├── control
│   └── tareas.js
├── css
│   └── estilos.css
├── index.html
├── README.md
├── test
└── views
    ├── menu_vertical.html
    └── pizarra.html

Algunas operaciones que tiene la aplicación son:

	Nueva tarea
	Actualizar tarea
	Eliminar tarea
